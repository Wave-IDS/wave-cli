# wave-cli

Command line interface for Wave to ease development

```
$ vim .env
$ wave-cli
Wave CLI
root@wave-cli> help

Available commands:

        help
        exit

        new
                collector <name>
                user <username> <password>
        get
                collectors
                users
                tls
        update
                password <old> <new>
                assignpassword <username> <password>
                username <name>
                tls <cert file> <key file>
        delete
                collector <name>
                user <username>
        stream
                visualizer

root@wave-cli> exit
$
```
